# SmokeBye Static Site

This directory contains the static pages intended for deployment to Cloudflare Pages:

- `/` homepage
- `/privacy`
- `/terms`

## Suggested Cloudflare Pages setup

- Framework preset: `None`
- Build command: leave empty
- Build output directory: `site`

## Recommended custom domain

- `smokebye.sixtree.net`

## Suggested App Store URLs

- Privacy Policy URL: `https://smokebye.sixtree.net/privacy`
- Terms of Use URL: `https://smokebye.sixtree.net/terms`
